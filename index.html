geojson2bbox

Convert GeoJSON to Overpass Turbo Bounding Box Format

info About This Tool

This tool helps you check the latest OSM data mapped using Tasking Manager projects via Overpass Turbo.

compare Key Differences from Tasking Manager Download

  • download TM Download: Gets all objects within the project area
  • tune This Tool: Allows custom filtering and modifications for specific needs

check_circle Tool Features

  • grid_on Extract GeoJSON grids from Tasking Manager projects to determine BBOX
  • filter_list Select specific parameters (e.g., buildings only, roads, waterways)
  • schedule Set time filters to get objects mapped in OSM after a specific date
  • code Choose between JSON or XML output formats
  • open_in_new Copy generated queries and open directly in Overpass Turbo

map Typical Workflow

  1. Get GeoJSON grid from your Tasking Manager project using the download link below
  2. Paste/upload it here to calculate the Bounding Box (BBOX)
  3. Configure filters (date, element type, etc.)
  4. Generate Overpass Turbo query
  5. Copy and run in Overpass Turbo to verify recent mapping

get_app Download AOI from Tasking Manager

Get GeoJSON from any Tasking Manager project. Just replace PROJECT_ID with your project number in this URL:

https://tasking-manager-production-api.hotosm.org/api/v2/projects/[PROJECT_ID]/queries/aoi/?as_file=true

Example for project 36867: https://tasking-manager-production-api.hotosm.org/api/v2/projects/36867/queries/aoi/?as_file=true

help How to use:

  1. Replace PROJECT_ID with your project number (found in the project URL)
  2. Open the URL in your browser
  3. Save the GeoJSON file
  4. Upload it below using "Upload File" or copy-paste the content
